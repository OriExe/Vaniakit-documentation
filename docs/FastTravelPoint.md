# FastTravelPoint class
A class that defines a fast travel point in your level. Can be saved in a Json file by pressing the button in the editor (Missing image).
If you have at least one FastTravel point defined and saved in the json file by clicking the `Save this point in Json` button then you'll be able to fast travel to different points and set those areas.

## Events
#### `onPlayerLoadedHere()`

Triggers when the player has sucessfully loaded at a specific fast travel point

## Accessible Variables (In Unity Editor)
#### `pointName`
Defines the name of the specific that will be shown to player. You may want to name this to a specific part of the world like `The Jungle`
